## Instalação e uso

### API (Back-end)

```bash
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/martins-rafael/proffy.git
# ou use a opção de download.

# Entre na pasta server 
$ cd proffy/server

# Instale as dependências
$ yarn install

# Rode o servidor
$ yarn start
```

<br>

### Web (Front-end)

```bash
# Entre na pasta web com 
$ cd proffy/web

# Instale as dependências
$ yarn install

# Rode o aplicação
$ yarn start
```
